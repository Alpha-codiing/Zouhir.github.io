Affichage Météo Dynamique sur Écran

Contexte & Objectifs

Création d'un script en Python pour récupérer des données météorologiques actuelles et les afficher de manière claire et attractive sur un écran. Le projet met l'accent sur l'interrogation d'API, le traitement de données et l'affichage sur interface graphique (ou un écran externe piloté par un microcontrôleur).

Fonctionnalités Clés

  • Interrogation API :Utilisation de la librairie `requests` de Python pour se connecter à une API météorologique (ex: OpenWeatherMap).
  • Traitement JSON : Analyse des données renvoyées au format JSON et extraction des informations pertinentes (température, humidité, prévisions).
  • Affichage Dynamique : Utilisation d'une librairie d'interface utilisateur (ex: tkinter, ou pilotage d'un écran OLED) pour la visualisation des données.

Technologies Maîtrisées

Python API JSON Parsing Interface Utilisateur (Tkinter/Microcontrôleur)
